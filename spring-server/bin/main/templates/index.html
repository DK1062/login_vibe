AI Health Check

건강검진 데이터 기반 비만 예측 서비스

키와 몸무게 정보를 입력하면 Flask AI 서버가 비만 여부를 예측하고, Spring Boot 화면에서 결과를 확인할 수 있습니다.

AI 예측 시작 게시판 보기

서비스 흐름

  1. 사용자가 키와 몸무게를 입력합니다.
  2. Spring Boot가 Flask API로 요청합니다.
  3. AI 모델 예측 결과를 화면에 출력합니다.

회원/로그인

Spring Security 기반 로그인과 회원 권한 관리를 제공합니다.

게시판

로그인 사용자는 게시글을 작성하고 본인 글을 관리할 수 있습니다.

AI 비만 예측

Flask 머신러닝 서버와 REST API로 연동해 예측 결과를 제공합니다.
